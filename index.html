§ 01 — Buniyaad / The Foundation

What is a Metric Space?

Kya aapne kabhi socha: "doori" ka matlab asal mein kya hai? Metric space usi sawaal ka jawab hai.

Roz ki zindagi mein hum doori mapte hain — shahr ki sadkon par, jeb mein rakhe ruler se, ya flight path se. Lekin mathematicians ne socha: in sab mein kya cheez common hai? Unhone chhaan ke sirf 4 rules nikale — aur unhe "metric axioms" kaha.

🇮🇳 Samjhao Simply Socho ek shahar ka map. Do jagah ke beech "doori" hoti hai. Yahi concept generalize karke kisi bhi set par lagana — yahi hai metric space ka kaam!
Metric Space — Definition

A metric space is a pair (X, d) where X is a non-empty set and d : X × X → ℝ is a function called the metric (or distance function), satisfying the following four axioms for all x, y, z ∈ X:

  • M1
    Non-negativity (Gairamaujoodiat nahin): d(x, y) ≥ 0
    and d(x, y) = 0 ⟺ x = y
    Doori kabhi negative nahi hoti. Zero doori ka matlab: dono ek hi jagah hain.
  • M2
    Symmetry (Samacharchita): d(x, y) = d(y, x)
    Delhi se Mumbai ki doori = Mumbai se Delhi ki doori. Simple!
  • M3
    Triangle Inequality (Tribhuj asamanta): d(x, z) ≤ d(x, y) + d(y, z)
    Seedha jaana, rasate se jaane se kabhi zyada lambi nahi hoti. Koi shortcut nahi ban sakta!

Kisi bhi point ko drag karein — notice karein d(x,z) ≤ d(x,y) + d(y,z) hamesha sach rahega.
Pseudo-Metric Space

A pseudo-metric (ya semi-metric) is the same as a metric, except we drop the condition d(x,y) = 0 ⟹ x = y. So two different points can be at distance zero.

Example: d(f, g) = sup|f(x) − g(x)| on a set of functions where two functions may agree except on a set of measure zero.

Diameter of a Set

The diameter of a non-empty subset A ⊆ X is:

diam(A) = sup { d(x, y) : x, y ∈ A }

If diam(A) < ∞, we say A is bounded. Think of it as the "widest" the set gets.

∎
§ 02 — Metric ke Prakar / Types of Metrics

The Menagerie of Metrics

Ek hi set par alag-alag metrics — har ek doori ko alag nazar se dekhti hai.

Click a metric below to see how its "ball" (set of points within distance r) looks. Same set, completely different shapes!

Euclidean (Usual)

d(x,y) = √(Σ(xᵢ−yᵢ)²)

Ruler se seedha measure karein. "Jaise crow uda tha" wali doori. Most common metric on ℝⁿ.

Manhattan (Taxicab)

d(x,y) = Σ|xᵢ−yᵢ|

Shahar ki sadkon par chalte chalte. Sirf horizontal/vertical movement. L¹ metric.

Chebyshev (L∞)

d(x,y) = max|xᵢ−yᵢ|

Chess mein raja ka move. Sabse badi coordinate difference lena. L∞ metric.

Discrete Metric

d(x,y) = 0 (x=y), 1 (x≠y)

Ya same ho, ya ek unit door. Kisi bhi set par kaam karta hai — even non-numeric sets!

Usual Metric on ℝ

The most important example: on the real line ℝ, the usual metric is:

d(x, y) = |x − y|

This satisfies all four axioms. Almost all of real analysis is built on this single metric!

Metric on ℝⁿ

On ℝⁿ, the three most important metrics (all valid) are:

NameFormulaBall Shape
Euclidean d₂ √(Σ(xᵢ−yᵢ)²) Circle / Sphere
Manhattan d₁ Σ|xᵢ−yᵢ| Diamond / Octahedron
Chebyshev d∞ max|xᵢ−yᵢ| Square / Cube
§ 03 — Topology

Open & Closed Sets, Neighbourhoods

Metric se topology janam leti hai — "nazdeeiki" ka ek deeper idea, bina measure ke.

🇮🇳 Samjhao Simply Socho ek point ko apne ghar ki tarah. "Open set" ek aisi mohalla hai jahan har ghar ke aas-paas thodi jagah hoti hai — matlab koi bhi "edge" par nahi rehta. "Closed set" mein edge ke point bhi included hain.
Open Ball / Neighbourhood

The open ball (ya neighbourhood) of radius r > 0 centred at x ∈ X:

B(x, r) = { y ∈ X : d(x, y) < r }

Note: strict inequality (<), so the boundary is not included. This is like an open interval in ℝ.

Interior Point

A point x ∈ A is an interior point of A if there exists r > 0 such that B(x, r) ⊆ A.

The set of all interior points of A is the interior A° = Int(A).

Open Set

A set U ⊆ X is open if every point of U is an interior point:

∀ x ∈ U, ∃ r > 0 : B(x, r) ⊆ U

Equivalently: U = Int(U).

Limit Point (Accumulation Point / Adherent Point)

A point x ∈ X is a limit point (accumulation point) of A if every neighbourhood of x contains a point of A different from x:

∀ r > 0, (B(x, r) \ {x}) ∩ A ≠ ∅

An adherent point (or point of closure) is slightly weaker — every neighbourhood intersects A (the point itself may be the intersection).

Closed Set & Closure

A set F ⊆ X is closed if it contains all its limit points. Equivalently, F is closed iff its complement X \setminus F is open.

The closure of A is the smallest closed set containing A:

Ā = A ∪ {limit points of A}

Key Theorems on Open / Closed Sets
  • ∅ and X are both open and closed ("clopen").
  • Arbitrary unions of open sets are open.
  • Finite intersections of open sets are open.
  • Arbitrary intersections of closed sets are closed.
  • Finite unions of closed sets are closed.
Proof: Arbitrary Union of Open Sets is Open

Let {Uα} be any collection of open sets. Let U = ∪ Uα and let x ∈ U.

Then x ∈ Uα for some α. Since Uα is open, ∃ r > 0 with B(x,r) ⊆ Uα ⊆ U.

Since x was arbitrary, U is open. ∎

Interior of a Set

The interior A° is the largest open set contained in A. It equals the union of all open sets contained in A.

A° = ∪ { U : U is open, U ⊆ A }
Bolzano–Weierstrass Theorem

Every bounded infinite subset of ℝⁿ has at least one limit point (accumulation point).

Intuition (Hindi): Agar ek bounded region mein infinitely many points hain, toh kuch points ek dusre ke bahut karib honge — unka ek "gathering spot" zaroor hoga!

Proof Sketch: Bolzano–Weierstrass (ℝ)

Let S ⊆ ℝ be bounded and infinite. Since bounded, S ⊆ [a, b]. Bisect [a,b] into two halves — at least one half contains infinitely many points of S. Bisect again, and again... This gives a nested sequence of intervals with lengths → 0. Their intersection is a single point, which is a limit point of S. ∎

Continuity — ε-δ Definition

A function f : (X, d) → (Y, d') is continuous at x₀ if:

∀ ε > 0, ∃ δ > 0 : d(x, x₀) < δ ⟹ d'(f(x), f(x₀)) < ε

And f is continuous if it is continuous at every point. Equivalently, preimages of open sets are open.

§ 04 — Sequences

Convergence & Cauchy Sequences

Kya ek sequence ka "destination" hota hai? Convergence isi sawaal ka jawaab hai.

🇮🇳 Samjhao Simply Socho ek ant ek line par chal rahi hai. Agar woh ek hi jagah ki taraf milti ja rahi hai — toh woh "converge" kar rahi hai. Agar woh darr rahi hai to — "diverge".
Convergent Sequence

A sequence (xₙ) in a metric space (X, d) converges to L if:

∀ ε > 0, ∃ N ∈ ℕ : n > N ⟹ d(xₙ, L) < ε

We write xₙ → L or lim xₙ = L. The point L is the limit.

Uniqueness of Limits

In a metric space, a convergent sequence has a unique limit.

Proof

Suppose xₙ → L and xₙ → M. Then for any ε > 0:

d(L, M) ≤ d(L, xₙ) + d(xₙ, M) < ε/2 + ε/2 = ε

Since ε was arbitrary, d(L,M) = 0, so L = M. ∎

Bounded Sequence

A sequence (xₙ) is bounded if the set {xₙ : n ∈ ℕ} is bounded, i.e., ∃ M > 0 such that d(xₙ, x₁) ≤ M for all n.

Convergent ⟹ Bounded

Every convergent sequence is bounded. (Converse is NOT generally true!)

Proof

Let xₙ → L. Take ε = 1. Then ∃ N such that for n > N, d(xₙ, L) < 1.

Let M = max{d(x₁,L), d(x₂,L), ..., d(xₙ,L), 1} + 1. Then d(xₙ, L) ≤ M for all n. ∎

Cauchy Sequence

A sequence (xₙ) is Cauchy if the terms get arbitrarily close to each other:

∀ ε > 0, ∃ N ∈ ℕ : m, n > N ⟹ d(xₘ, xₙ) < ε

Key intuition: Cauchy sequence ke terms ek-dusre ke paas aate jaate hain — chahe unka limit X mein ho ya na ho!

Important: In ℝ or ℝⁿ, every Cauchy sequence converges. But in general metric spaces, a Cauchy sequence may NOT converge within the space (e.g., ℚ with usual metric).

Convergent ⟹ Cauchy

Every convergent sequence is a Cauchy sequence.

Proof

Let xₙ → L. Given ε > 0, choose N such that n > N ⟹ d(xₙ, L) < ε/2. Then for m, n > N:

d(xₘ, xₙ) ≤ d(xₘ, L) + d(L, xₙ) < ε/2 + ε/2 = ε ∎

§ 05 — Sampurnata / Completeness

Complete Metric Spaces

Kuch spaces mein har Cauchy sequence ka ek "ghar" hota hai. Aise spaces ko "complete" kehte hain.

Complete Metric Space

A metric space (X, d) is complete if every Cauchy sequence in X converges to a point in X.

Examples:

  • ℝ with usual metric — complete ✓
  • ℚ with usual metric — NOT complete (√2 is a limit not in ℚ)
  • ℝⁿ — complete ✓
  • Any discrete metric space — complete ✓
  • Open interval (0,1) with usual metric — NOT complete
🇮🇳 Samjhao Simply ℚ ek "incomplete" space hai — socho 1, 1.4, 1.41, 1.414, 1.4142, ... yeh sequence ℚ mein Cauchy hai, lekin iski limit √2 hai jo ℚ mein hi nahi! Matlab sequence ek aisi jagah jaana chahti hai jo exist hi nahi karti.
Completeness of ℝ (Completeness Axiom)

Every Cauchy sequence of real numbers converges to a real number. This is a fundamental axiom / theorem of real analysis.

Nested Interval Theorem

If [a₁, b₁] ⊇ [a₂, b₂] ⊇ ... is a nested sequence of closed bounded intervals with (bₙ − aₙ) → 0, then there is exactly one point in all of them:

∩ₙ [aₙ, bₙ] = {x} for some x ∈ ℝ
§ 06 — Connected & Disconnected Sets

Separation, Connectedness & Continuity

Kya ek set "ek tukda" hai, ya "toot ke" alag-alag parts mein bata hua hai?

🇮🇳 Samjhao Simply Socho ek continuous line vs. do alag points. Ek "connected" hai, doosra "disconnected". Ek kadam bhi kaafi hai alag karne ke liye!
Separated Sets

Two sets A, B ⊆ X are separated if:

Ā ∩ B = ∅ and A ∩ B̄ = ∅

Neither set contains a limit point of the other. They are truly "far apart" even if not disjoint as sets.

Connected & Disconnected Sets

A set E ⊆ X is disconnected if it can be written as E = A ∪ B where A and B are non-empty separated sets.

A set is connected if it is NOT disconnected — it cannot be split into two separated non-empty parts.

Examples
  • ℝ is connected. Any interval [a,b], (a,b) is connected.
  • ℝ \ {0} = (-∞,0) ∪ (0,∞) is disconnected.
  • ℚ is totally disconnected — no non-trivial connected subsets.
  • Any single point set {x} is connected.
Continuous Image of a Connected Set is Connected

If f : X → Y is continuous and E ⊆ X is connected, then f(E) is connected in Y.

Proof

Suppose f(E) = A ∪ B with A, B separated. Let G = f⁻¹(A) ∩ E and H = f⁻¹(B) ∩ E. Then G, H are non-empty (if A, B non-empty), G ∪ H = E, and one can show G, H are separated (using continuity to pull back the separation). This contradicts E being connected. ∎

Intermediate Value Theorem (IVT) — Metric Space Version

If f : X → ℝ is continuous and X is connected, and f(a) < c < f(b), then ∃ x ∈ X with f(x) = c.

Intuition: Ek connected set pe continuous function "jump" nahi kar sakta — beech ki har value leni padti hai!

§ 07 — Samabhava / Compactness

Compact Sets & Uniform Continuity

Compactness ek powerful property hai — "bounded aur closed hona" ka ek generalization.

🇮🇳 Samjhao Simply Compact set ek aisi cheez hai jahan "har chiz limited jagah mein fit ho jaye". Jaisa ek finite-size box — koi cheez bahar nahi ja sakti aur nahi hi infinitely bhar sakti!
Open Cover & Compactness

An open cover of a set K ⊆ X is a collection {Uα} of open sets such that K ⊆ ∪ Uα.

A set K is compact if every open cover has a finite subcover — i.e., finitely many of the Uα already cover K.

Heine-Borel Theorem (in ℝⁿ)

A subset K ⊆ ℝⁿ is compact if and only if it is closed and bounded.

Note: This holds specifically in ℝⁿ. In general metric spaces, compact ⟹ closed and bounded, but not necessarily the converse.

Proof Sketch: [a,b] is Compact (Heine-Borel, 1D)

Let {Uα} cover [a,b]. Suppose no finite subcover exists. Bisect [a,b] — at least one half has no finite subcover. Bisect again, and again... We get a nested sequence of closed intervals [aₙ, bₙ] with lengths → 0 and no finite subcover. By the Nested Interval Theorem, there exists x ∈ ∩[aₙ, bₙ]. Then x ∈ Uα for some α, so ∃ r > 0 with (x-r, x+r) ⊆ Uα. But for large n, [aₙ, bₙ] ⊆ (x-r, x+r) ⊆ Uα — a single set covers the interval. Contradiction! ∎

Compact Sets are Closed and Bounded

In any metric space, every compact set is closed and bounded.

Proof: Compact ⟹ Bounded

Fix x₀ ∈ K. The collection {B(x₀, n) : n ∈ ℕ} is an open cover of K. By compactness, finitely many cover K: K ⊆ B(x₀, N) for some N. So K is bounded. ∎

Continuous Image of Compact Set is Compact

If f : X → Y is continuous and K ⊆ X is compact, then f(K) is compact in Y.

Proof

Let {Vα} be an open cover of f(K). Then {f⁻¹(Vα)} is an open cover of K (by continuity). By compactness of K, finitely many suffice: K ⊆ f⁻¹(Vα₁) ∪ ... ∪ f⁻¹(Vαₙ). Then f(K) ⊆ Vα₁ ∪ ... ∪ Vαₙ. ∎

Uniform Continuity

A function f : (X,d) → (Y,d') is uniformly continuous if:

∀ ε > 0, ∃ δ > 0 : d(x, y) < δ ⟹ d'(f(x), f(y)) < ε (for ALL x, y)

Key difference from continuity: In uniform continuity, the same δ works for every point. In ordinary continuity, δ can depend on the point x₀.

🇮🇳 Samjhao Simply Ordinary continuity: har jagah "thoda zoom in karo" toh function smooth lagta hai — lekin kahin zyada zoom karna pad sakta hai, kahin kam. Uniform continuity: ek hi "zoom level" pe poori line smooth lagti hai!
Compact ⟹ Uniform Continuity

If f : K → Y is continuous and K is compact, then f is uniformly continuous on K.

Proof Sketch

Given ε > 0, for each x ∈ K choose δ(x) by continuity. The balls {B(x, δ(x)/2)} cover K. By compactness, finitely many cover: B(x₁, δ₁/2), ..., B(xₙ, δₙ/2). Let δ = min(δᵢ/2). One can verify this δ works uniformly for all pairs. ∎

Extreme Value Theorem

If f : K → ℝ is continuous and K is compact, then f attains its maximum and minimum on K.

Intuition: Ek continuous function ek compact set par hamesha apni max aur min value "reach" karti hai — na ki bas "approach" karti hai!
