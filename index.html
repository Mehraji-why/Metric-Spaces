Foundation

What Is A Metric Space?

A metric space is one of the most fundamental structures in mathematics. It gives us a rigorous way to define “distance” between objects.

d(x,y) ≥ 0
d(x,y) = d(y,x)
d(x,z) ≤ d(x,y) + d(y,z)
Common Metrics

Different Metrics Create Different Geometries

Euclidean Metric

d₂(x,y)=√Σ(xᵢ-yᵢ)²

The familiar straight-line distance from geometry.

Manhattan Metric

d₁(x,y)=Σ|xᵢ-yᵢ|

Distance measured along grid-like movement.

Chebyshev Metric

d∞(x,y)=max|xᵢ-yᵢ|

Distance determined by the largest coordinate change.

Topology

Open Sets & Neighborhoods

Open balls define the topology of a metric space. From them arise continuity, convergence, compactness, and modern analysis itself.

B(x,r) = { y ∈ X : d(x,y) < r }
Sequences

Convergence & Cauchy Sequences

Convergent

Terms approach a fixed limit point.

Cauchy

Terms become arbitrarily close to one another.

Bounded

All terms remain inside a finite region.
