§ 01 — The Foundation

What is a Metric Space?

Before the definition, a question: what does "distance" actually mean?

In everyday life we measure distances — across cities, on rulers, through flight paths. But a mathematician would ask something deeper: what is the bare minimum a "distance function" must satisfy to be trustworthy? The answer is exactly three axioms. Any function satisfying these three rules is called a metric, and the set it operates on is a metric space.

Etymology "Metric" — from Greek metron (μέτρον), meaning "measure." The same root gives us meter, geometry, symmetry.
🧭 Intuition First

Imagine a map of cities. Any two cities have a "distance." Now generalise — forget cities, take any set of objects, and define a "distance" between any two that obeys three natural rules. That's all a metric space is: a set + a trustworthy notion of closeness.

Definition
Metric Space

A metric space is a pair (X, d) where X is a non-empty set and d : X × X → ℝ is a function (the metric or distance function) satisfying, for all x, y, z ∈ X:

  • M1
    Non-negativity & Identity:   d(x,y) ≥ 0, and d(x,y) = 0 ⟺ x = y Distance is never negative. Only a point is at distance zero from itself.
  • M2
    Symmetry:   d(x,y) = d(y,x) Distance from A to B equals distance from B to A — direction doesn't matter.
  • M3
    Triangle Inequality:   d(x,z) ≤ d(x,y) + d(y,z) Going via a third point can never create a shortcut. The direct path is always shortest (or equal).
▸ Interactive — Triangle Inequality (drag any point)
Drag any of the three points. The triangle inequality d(x,z) ≤ d(x,y) + d(y,z) always holds — watch it live.
∿

Pseudo-Metric Space

A pseudo-metric relaxes M1: we allow d(x,y) = 0 even when x ≠ y. Two distinct points may be "zero apart."

Example
Pseudo-Metric on Integrable Functions

On Riemann-integrable functions on [0,1]:

d(f, g) = ∫₀¹ |f(x) − g(x)| dx

If f and g differ only on a finite set (measure zero), then d(f,g) = 0 yet f ≠ g. So this is a pseudo-metric, not a true metric.

Diameter of a Set

Definition
Diameter & Bounded Set

Let (X, d) be a metric space and A ⊆ X, A ≠ ∅. The diameter of A is:

diam(A) = sup { d(x,y) : x, y ∈ A }

If diam(A) < ∞, then A is bounded. For a singleton A = {x}, we have diam(A) = 0.

Examples
Diameter Calculations

In ℝ (usual metric): diam([a,b]) = b − a.

Open ball of radius r in ℝⁿ: diam(B(x,r)) ≤ 2r, with equality.

Discrete metric on any set X with |X| ≥ 2: diam(X) = 1.

Can d(x,y) = (x−y)² define a metric on ℝ?
No. It fails the triangle inequality. Take x=0, y=2, z=1: d(0,2) = 4, but d(0,1)+d(1,2) = 1+1 = 2 < 4. The triangle inequality is violated.
§ 02 — Common Metrics

Standard Metric Spaces

A single set can carry many different metrics — each one creates a different geometry, different balls, different notion of closeness.

Click any metric card to see its unit ball visualised interactively below.

Euclidean Metric

d₂(x,y) = √(Σᵢ(xᵢ−yᵢ)²)

Straight-line distance. The geometry we inherit from physical intuition. Standard metric on ℝⁿ.

Manhattan / Taxicab

d₁(x,y) = Σᵢ|xᵢ−yᵢ|

City-grid distance — only horizontal and vertical movement allowed. The ℓ¹ norm.

Chebyshev / ℓ∞

d∞(x,y) = maxᵢ|xᵢ−yᵢ|

A chess king's movement distance — maximum coordinate difference. The ℓ∞ norm.

Discrete Metric

d(x,y) = 0 if x=y, 1 otherwise

Defined on any set. Any two distinct points are exactly distance 1 apart. Trivial but conceptually essential.

Usual Metric on ℝ

Definition
Usual Metric

The standard metric on ℝ: d(x,y) = |x − y|.

Proof
Verification of Metric Axioms
  • M1: |x−y| ≥ 0 and |x−y| = 0 ⟺ x = y — property of absolute value.
  • M2: |x−y| = |y−x| — since |−a| = |a|.
  • M3: |x−z| = |(x−y)+(y−z)| ≤ |x−y|+|y−z| — triangle inequality for |·|.

Lᵖ Metrics

Definition
Lᵖ Metric on ℝⁿ

For p ≥ 1:

dₚ(x,y) = ( Σᵢ₌₁ⁿ |xᵢ−yᵢ|ᵖ )^(1/p)

Special cases: p=1 → Manhattan; p=2 → Euclidean; p→∞ → Chebyshev. The triangle inequality for these is Minkowski's Inequality.

Discrete Metric — Deep Dive

Theorem
Properties of Discrete Metric Space

In (X, d_disc): every subset is both open and closed; every convergent sequence is eventually constant; if X is infinite, it is not compact; it is always complete.

⚠ Common Mistake

Students often think "open" and "closed" are opposites — a set can be neither (e.g. a half-open interval), or both (clopen). The discrete metric makes every set clopen, which is a great counterexample to keep in mind.

§ 03 — Topology of Metric Spaces

Open Sets, Closed Sets & Neighbourhoods

Open and closed are not opposites — they are complementary ideas built from the notion of "ball."

Etymology "Topology" — from Greek topos (τόπος, "place") + logos (λόγος, "study"). The study of space itself.
Definition
Open Ball

Let (X,d) be a metric space, x ∈ X, r > 0. The open ball of radius r centred at x:

B(x, r) = { y ∈ X : d(x, y) < r }

And the closed ball: B̄(x, r) = { y ∈ X : d(x, y) ≤ r }.

Definition
Open Set

A set G ⊆ X is open if for every x ∈ G, there exists r > 0 such that B(x, r) ⊆ G.

Intuition: Every point has breathing room — a small ball around it that stays inside G. No point is "on the edge."

Theorem
Properties of Open Sets
  1. ∅ and X are both open.
  2. Arbitrary unions of open sets are open.
  3. Finite intersections of open sets are open.

Warning: Infinite intersections of open sets need not be open. E.g. ∩ₙ (−1/n, 1/n) = {0}, which is closed.

Definition
Closed Set

A set F ⊆ X is closed if its complement Fᶜ = X \ F is open — equivalently, F contains all its limit points.

▸ Interactive — Open vs Closed Balls
Open ball: boundary NOT included. Points inside have "breathing room."
∿

Interior, Closure, Boundary

Interior — int(A)

Largest open set contained in A. A point x ∈ int(A) if some open ball around x lies entirely in A.

Closure — Ā

Smallest closed set containing A. Ā = A ∪ (limit points of A).

Boundary — ∂A

∂A = Ā \ int(A). Every open ball around a boundary point meets both A and Aᶜ.

Dense Set

A is dense in X if Ā = X. Equivalently, every open ball in X meets A. (ℚ is dense in ℝ.)

Theorem
Characterisation of Closure

For any set A ⊆ X:

Ā = { x ∈ X : B(x,r) ∩ A ≠ ∅ for all r > 0 }

That is, x ∈ Ā if and only if every open ball around x hits A.

§ 04 — Points & Their Types

Interior, Limit & Adherent Points

Every point in a metric space has a precise relationship to any given set. Learning to classify points is the first step toward analysis.

Interior Point

x is an interior point of A if ∃ r > 0: B(x,r) ⊆ A. The point has an open neighbourhood entirely within A.

Limit Point

x is a limit (accumulation) point of A if every ball B(x,r) contains a point of A other than x itself.

Adherent Point

x is an adherent point of A if every B(x,r) meets A (x itself counts). So Ā = set of all adherent points.

Isolated Point

x ∈ A is isolated if ∃ r > 0: B(x,r) ∩ A = {x}. No other points of A nearby.

⚠ Students Often Confuse This

A limit point of A need NOT be in A. For example, 0 is a limit point of (0,1) but 0 ∉ (0,1). Conversely, an isolated point IS in A but is NOT a limit point. Keep these separate.

Theorem
F is Closed ⟺ F Contains All Its Limit Points

A set F ⊆ X is closed if and only if it contains all its limit points (equivalently, F' ⊆ F where F' denotes the derived set).

Proof
Proof

(⟹) F closed: Let x be a limit point of F. Suppose x ∉ F; then x ∈ Fᶜ which is open. So ∃ r>0: B(x,r) ⊆ Fᶜ, meaning B(x,r) ∩ F = ∅. But x is a limit point — contradiction.

(⟸) F contains all limit points: Suppose x ∉ F. Then x is not a limit point of F, so ∃ r>0: B(x,r) ∩ F = ∅ (or only contains x, but x ∉ F). Thus B(x,r) ⊆ Fᶜ. Since x ∈ Fᶜ was arbitrary and has an open neighbourhood in Fᶜ, Fᶜ is open, so F is closed. □

Examples
Limit Points — Common Examples

A = (0,1) in ℝ: Every point of [0,1] is a limit point of A. Isolated points: none.

A = {1/n : n ∈ ℕ} in ℝ: Every point 1/n is isolated. The only limit point is 0, which is NOT in A.

A = ℚ in ℝ: Every real number is a limit point of ℚ (rationals are dense). No isolated points.

§ 05 — Sequences

Convergent, Cauchy & Bounded Sequences

A sequence is a journey. The question is: does it arrive somewhere?

Etymology "Converge" — from Latin convergere: com- ("together") + vergere ("to incline"). To incline together toward a point.
Definition
Convergent Sequence

A sequence (xₙ) in (X,d) converges to x ∈ X if:

∀ ε > 0, ∃ N ∈ ℕ such that n ≥ N ⟹ d(xₙ, x) < ε

We write xₙ → x or lim xₙ = x. The limit, if it exists, is unique.

Definition
Cauchy Sequence

A sequence (xₙ) is a Cauchy sequence if:

∀ ε > 0, ∃ N ∈ ℕ such that m, n ≥ N ⟹ d(xₙ, xₘ) < ε

Key insight: The terms get arbitrarily close to each other — whether or not a limit exists in the space.

🔑 Key Distinction

Convergent: terms approach a fixed point in X. Cauchy: terms approach each other. In a complete space these are equivalent — but in ℚ, a Cauchy sequence might "want" to converge to √2, which doesn't live in ℚ.

▸ Sequence Visualiser — Convergent vs Cauchy
Convergent sequence 1/n → 0: every term after index N lies inside the ε-band around 0.
Theorem
Convergent ⟹ Cauchy

Every convergent sequence is a Cauchy sequence. The converse holds if and only if the space is complete.

Proof
Proof

Let xₙ → x. Given ε > 0, choose N so that n ≥ N ⟹ d(xₙ,x) < ε/2.

For m, n ≥ N: d(xₙ,xₘ) ≤ d(xₙ,x) + d(x,xₘ) < ε/2 + ε/2 = ε. □

Example
Cauchy but Not Convergent in ℚ

In (ℚ, |·|), the sequence of decimal approximations of √2:

x₁ = 1.4, x₂ = 1.41, x₃ = 1.414, x₄ = 1.4142, …

This is Cauchy — terms get arbitrarily close. But √2 ∉ ℚ, so the sequence has no limit in ℚ. This proves ℚ is not complete.

Bounded Sequences

Definition
Bounded Sequence

A sequence (xₙ) in (X,d) is bounded if there exists M > 0 and a point a ∈ X such that d(xₙ,a) ≤ M for all n.

Theorem
Convergent ⟹ Bounded

Every convergent sequence is bounded. The converse fails: (−1)ⁿ is bounded but not convergent in ℝ.

§ 06 — Completeness

Complete Metric Spaces & Bolzano–Weierstrass

Completeness is the property that says: "our space has no hidden gaps." Every Cauchy sequence already has its destination inside.

Definition
Complete Metric Space

A metric space (X,d) is complete if every Cauchy sequence in X converges to a limit that lies in X.

🧠 Intuition — Completeness

ℝ is complete: if terms squeeze together, their limit is real. ℚ is not complete: a Cauchy sequence can "want" to converge to an irrational, which isn't there. Complete = no missing holes; no sequence can fall through the gaps.

SpaceComplete?Reason
ℝ (real numbers)✓ YesEvery Cauchy sequence converges in ℝ
ℚ (rationals)✗ NoCauchy sequences can converge to irrationals
ℝⁿ (Euclidean)✓ YesCompleteness coordinate-by-coordinate
(0,1) open interval✗ NoSequence 1/n → 0 ∉ (0,1)
[0,1] closed interval✓ YesClosed subset of complete space is complete
Discrete metric space✓ YesAll Cauchy sequences are eventually constant
Theorem
Closed Subsets of Complete Spaces

A subset F of a complete metric space (X,d) is complete (with induced metric) if and only if F is closed in X.

Proof
Proof (both directions)

(⟹) F complete ⟹ F closed: Let x be a limit point of F; choose (xₙ) in F with xₙ→x. Convergent ⟹ Cauchy. F complete ⟹ limit lies in F. So x ∈ F and F is closed.

(⟸) F closed ⟹ F complete: Let (xₙ) be Cauchy in F. It is also Cauchy in X. X is complete, so xₙ→x for some x ∈ X. F is closed and xₙ ∈ F, so x ∈ F. Hence the Cauchy sequence converges in F. □

∿

Bolzano–Weierstrass Theorem

Theorem
Bolzano–Weierstrass (ℝⁿ)

Every bounded sequence in ℝⁿ has a convergent subsequence.

🧭 Intuition

Squeeze infinitely many points into a bounded region — they must cluster somewhere. Like packing infinite guests into a finite hotel: at least one room has infinitely many assigned to it. That cluster point is the limit of a subsequence.

Proof Sketch
Bisection Argument (for ℝ)

Let (xₙ) ⊆ [a,b]. Apply the bisection method:

  1. Bisect [a,b] into two halves. At least one contains infinitely many terms — call it I₁.
  2. Bisect I₁. Choose the half with infinitely many terms — call it I₂.
  3. Continue: get nested closed intervals I₁ ⊇ I₂ ⊇ I₃ ⊇ … with lengths → 0.
  4. By the Nested Interval Property, ∩Iₖ = {x} for some x ∈ ℝ.
  5. Choose xₙₖ ∈ Iₖ for each k → this subsequence converges to x. □
§ 07 — Connectedness & Continuity

Separated, Connected & Continuous Maps

A connected set is one that cannot be torn in two. Understanding this unlocks the Intermediate Value Theorem and much of real analysis.

Definition
Separated Sets

Two subsets A, B of a metric space are separated if:

Ā ∩ B = ∅   and   A ∩ B̄ = ∅

Neither set contains a limit point of the other. They are pulled completely apart — not even sharing boundary points.

Definition
Connected & Disconnected

A set E ⊆ X is disconnected if E = A ∪ B for non-empty separated sets A, B. E is connected if no such separation exists — it is a single, indivisible piece.

▸ Connected vs Disconnected — Visual
A connected set — one continuous piece, impossible to separate without sharing boundary.
Theorem
Connected Subsets of ℝ are Intervals

A subset of ℝ (with usual metric) is connected if and only if it is an interval (including rays and ℝ itself).

Proof Sketch
Proof

(Interval ⟹ Connected): Suppose E is an interval and E = A∪B is a separation. Take a∈A, b∈B with a<b. Let c = sup(A∩[a,b]). Then c ∈ Ā. If c∈B then c ∈ A∩B̄ = ∅ — contradiction. If c∈A, since E is an interval [a,b]⊆E, so points just above c must lie in B, contradicting c∈A being in Ā of that part. Either way we reach contradiction.

(Not Interval ⟹ Disconnected): If E is not an interval, ∃ x,y∈E and z∉E with x<z<y. Then A=E∩(−∞,z) and B=E∩(z,∞) are non-empty, and one checks they are separated. □

∿

Continuity in a Metric Space

Definition
Continuous Function (ε-δ)

f: (X,d_X) → (Y,d_Y) is continuous at x₀ if:

∀ ε > 0, ∃ δ > 0 : d_X(x, x₀) < δ ⟹ d_Y(f(x), f(x₀)) < ε

f is continuous on X if it is continuous at every point of X.

Theorem
Four Equivalent Characterisations of Continuity

The following are equivalent for f: X → Y:

  1. f satisfies the ε-δ definition at every point.
  2. For every open set V in Y, the preimage f⁻¹(V) is open in X.
  3. For every closed set F in Y, the preimage f⁻¹(F) is closed in X.
  4. For every sequence xₙ → x in X, we have f(xₙ) → f(x) in Y. (Sequential continuity.)
Proof
ε-δ ⟺ Open Preimage

(⟹) Let V be open in Y, x ∈ f⁻¹(V). Then f(x) ∈ V, so ∃ε: B_Y(f(x),ε) ⊆ V. By continuity ∃δ: f(B_X(x,δ)) ⊆ B_Y(f(x),ε) ⊆ V. So B_X(x,δ) ⊆ f⁻¹(V). Since x was arbitrary, f⁻¹(V) is open.

(⟸) Given x₀ and ε>0, B_Y(f(x₀),ε) is open in Y. Its preimage f⁻¹(B_Y(f(x₀),ε)) is open in X and contains x₀. So ∃δ: B_X(x₀,δ) ⊆ f⁻¹(B_Y(f(x₀),ε)), i.e. d_X(x,x₀)<δ ⟹ d_Y(f(x),f(x₀))<ε. □

Corollary
Intermediate Value Theorem

If f: [a,b] → ℝ is continuous and f(a) < c < f(b), then ∃ x ∈ (a,b) with f(x) = c.

Proof: [a,b] is connected ⟹ f([a,b]) is connected ⟹ f([a,b]) is an interval containing f(a) and f(b) ⟹ c ∈ f([a,b]).

Uniform Continuity

Definition
Uniformly Continuous

f: X → Y is uniformly continuous if:

∀ ε > 0, ∃ δ > 0 : d_X(x,y) < δ ⟹ d_Y(f(x),f(y)) < ε   for ALL x,y ∈ X

Key difference: In ordinary continuity, δ may depend on both ε and the point x. In uniform continuity, one δ works globally for all x simultaneously.

Examples
Uniform vs Non-Uniform Continuity

f(x) = x² on ℝ: Continuous but not uniformly continuous. Near ∞, a tiny change in x causes a large change in x².

f(x) = x² on [0,1]: Uniformly continuous (compact domain — Heine-Cantor theorem applies).

f(x) = sin(x) on ℝ: Uniformly continuous, since |sin x − sin y| ≤ |x − y| by the Mean Value Theorem.

§ 08 — Compactness

Compact Sets & Uniform Continuity

Compactness is one of the most powerful ideas in analysis. It gives infinite sets the behaviour of finite ones — and that changes everything.

Etymology "Compact" — from Latin compactus, past participle of compingere: "to pack together tightly." A compact set cannot be spread out indefinitely.
Definition
Open Cover & Compact Set

An open cover of a set K is a collection {Gα} of open sets with K ⊆ ∪ Gα.

K is compact if every open cover of K has a finite subcover — finitely many of the Gα already cover K.

🧭 Intuition — Compactness

Imagine you need to cover a region with open disks (possibly infinitely many). Compact means: you can always pick finitely many of those disks that still cover the entire region. It's a "finite control" principle over infinite scenarios. Closed intervals in ℝ are compact. Open intervals are not — a cover by (1/n, 1) requires infinitely many sets.

Theorem
Heine–Borel Theorem (ℝⁿ)

A subset K of ℝⁿ is compact if and only if K is closed and bounded.

Proof Sketch
Both Directions

Compact ⟹ Closed: Let x ∉ K. For each y∈K, take balls B(y, rᵧ) and B(x, rᵧ) where rᵧ = d(x,y)/2. Compactness gives a finite subcover of K by B(yᵢ, rᵢ). Let δ = min(rᵢ); then B(x,δ)∩K = ∅. So Kᶜ is open.

Compact ⟹ Bounded: {B(0,n)} covers K; finite subcover gives K ⊆ B(0,N).

Closed + Bounded ⟹ Compact: K ⊆ [−M,M]ⁿ. Show [−M,M]ⁿ is compact (by nested bisection / Bolzano–Weierstrass); closed subset of compact is compact. □

Theorem
Three Equivalent Forms of Compactness (Metric Spaces)
  1. Cover compactness: Every open cover has a finite subcover.
  2. Sequential compactness: Every sequence in K has a subsequence converging to a point in K.
  3. Complete + Totally bounded: K is complete and every ε-net is finite.

In metric spaces, all three are equivalent — a beautiful confluence of different ideas.

Theorem
Continuous Image of a Compact Set is Compact

If f: X → Y is continuous and K ⊆ X is compact, then f(K) is compact in Y.

Proof
Proof

Let {Vα} be an open cover of f(K). By continuity, each f⁻¹(Vα) is open in X. These cover K. Compactness gives finitely many: f⁻¹(Vα₁),…,f⁻¹(Vαₙ) cover K. Then Vα₁,…,Vαₙ cover f(K). □

Corollary
Extreme Value Theorem

If f: K → ℝ is continuous and K is compact, then f attains its maximum and minimum values on K.

Proof: f(K) is compact ⟹ closed and bounded ⟹ sup f(K) and inf f(K) exist and are attained.

∿

Compactness and Uniform Continuity

Theorem
Heine–Cantor Theorem

If f: K → Y is continuous and K is compact, then f is uniformly continuous on K.

Proof
Proof

Given ε>0. By continuity at each x∈K, ∃ δ(x)>0: d(y,x)<δ(x) ⟹ d_Y(f(y),f(x))<ε/2.

The open balls {B(x, δ(x)/2) : x∈K} cover K. By compactness, finitely many x₁,…,xₙ give a subcover. Let δ = min(δ(xᵢ)/2).

If d(x,y)<δ, then x∈B(xᵢ, δ(xᵢ)/2) for some i, and d(y,xᵢ) ≤ d(y,x)+d(x,xᵢ) < δ+δ(xᵢ)/2 ≤ δ(xᵢ). So d_Y(f(x),f(xᵢ))<ε/2 and d_Y(f(y),f(xᵢ))<ε/2, giving d_Y(f(x),f(y))<ε. □

Is (0,1) compact in ℝ? What about [0,∞)?
(0,1) is not compact — it is bounded but not closed (0 and 1 are missing). The cover {(1/n, 1) : n∈ℕ} has no finite subcover.

[0,∞) is not compact — it is closed but not bounded. The cover {[0,n) : n∈ℕ} has no finite subcover.
