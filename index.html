§ 01 — The Axioms

What Is a Metric Space?

Distance is everywhere. But what truly makes something a "distance"?

Mathematicians distilled the essential properties of distance into three elegant axioms. A metric space is any set equipped with a function satisfying these axioms — no geometry required.

∀ x, y, z ∈ X
Metric Space

A metric space is a pair (X, d) where X is a set and d : X × X → ℝ is a function satisfying the axioms below for all x, y, z ∈ X.

  • M1
    Non-negativity: d(x,y) ≥ 0, and d(x,y) = 0 ⟺ x = y
    Distance is never negative. Zero distance means the points are identical.
  • M2
    Symmetry: d(x,y) = d(y,x)
    The distance from A to B equals the distance from B to A.
  • M3
    Triangle Inequality: d(x,z) ≤ d(x,y) + d(y,z)
    No shortcut exists — going via a third point is never cheaper than the direct path.

§ 02 — Examples

The Menagerie of Metrics

One set can carry many different metrics, each revealing different structure.

Select a metric below to see how its "ball" — the set of all points within distance r — looks in ℝ².

Euclidean

d(x,y) = √(Σ(xᵢ−yᵢ)²)

The familiar straight-line distance. "As the crow flies."

Manhattan

d(x,y) = Σ|xᵢ−yᵢ|

Grid-walking distance. Common in city planning.

Chebyshev

d(x,y) = max|xᵢ−yᵢ|

The king's move in chess. The dominant coordinate wins.

Discrete

d(x,y) = 0 if x=y, 1 otherwise

Points are either identical or maximally far. Works on any set.

§ 03 — Topology

Open & Closed Sets

Metric spaces give birth to topology — studying "nearness" without measurement.

Open Ball

The open ball of radius r centred at x ∈ X is

B(x, r) = { y ∈ X : d(x, y) < r }
Open Set

A set U ⊆ X is open if every point has "breathing room" — formally:

∀ x ∈ U, ∃ r > 0 : B(x, r) ⊆ U
▸ Open vs Closed — hover near the boundary
Hover near the dashed boundary to see the breathing-room ball.
Key Facts about Open Sets:
  • ∅ and X are both open (and both closed — "clopen").
  • Arbitrary unions of open sets are open.
  • Finite intersections of open sets are open.
  • A set F is closed if and only if its complement X \ F is open.

Continuity via Open Sets

A function f:(X,d)→(Y,d') is continuous iff preimages of open sets are open — which is equivalent to the classical ε-δ condition:

∀ ε > 0, ∃ δ > 0 : d(x,y) < δ ⟹ d'(f(x),f(y)) < ε

§ 04 — Sequences

Convergence

A sequence is a journey through the space. Does it settle somewhere?

Convergent Sequence

A sequence (x_n) converges to L ∈ X if

∀ ε > 0, ∃ N ∈ ℕ : n ≥ N ⟹ d(x_n, L) < ε

Eventually, every term lives inside any open ball around L.

Cauchy Sequence

A sequence is Cauchy if its terms become arbitrarily close to each other:

∀ ε > 0, ∃ N : m,n ≥ N ⟹ d(x_m, x_n) < ε

Every convergent sequence is Cauchy. The converse is precisely completeness.

§ 05 — Completeness

Complete Metric Spaces

A space is complete when it has no "holes" — no missing limit points.

The rationals ℚ are not complete — the sequence 3, 3.1, 3.14, 3.141, … is Cauchy but its limit π is irrational, hence absent from ℚ. The reals ℝ are the completion of ℚ.

Complete Metric Space

A metric space (X,d) is complete if every Cauchy sequence in X converges to a point in X.

Banach Fixed Point Theorem. Let (X,d) be a complete metric space and f:X→X a contraction: d(f(x),f(y)) ≤ k·d(x,y) for some k < 1. Then f has a unique fixed point x*, and the iteration x_{n+1} = f(x_n) converges to it regardless of the starting point.
